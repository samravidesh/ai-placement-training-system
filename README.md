
# AI-Powered Placement & Training Management System

An intelligent full-stack placement platform built using Node.js, PostgreSQL, and OpenAI.

##  Features

- JWT Authentication
- Profile Completion Tracker
- Aptitude Test Module
- Job Recommendation Engine
- 6-Month Smart Training Plan
- Resume-JD Semantic Matching
- RAG-based AI Chatbot

##  AI Integration

- OpenAI GPT
- Embeddings-based semantic search
- Retrieval-Augmented Generation (RAG)
- Personalized career guidance

##  Tech Stack

- Node.js
- Express
- PostgreSQL (Neon DB)
- OpenAI API
- JWT Authentication
